# アルティメットスイカゲーム

果物以外の様々なものでスイカゲームが遊べるように、生成AIを駆使して作成した自由度の高いゲームです。

## Run

```bash
git clone https://github.com/kuwabarbara/SuikaGame.git
cd suika-game
pip -r install requirements.txt
python server.py
npm install
npm run dev
```
